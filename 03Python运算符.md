# 逻辑运算符
+ not
+ and
+ or
注意：
+ 非布尔类型的值，如果作为真假来判定，一般都是非零即真，非空即真
+ 整个逻辑表达式的结果不一定只是True和False

例：
```python
print(0 and True) # 0
print(1 and 3) # 3
print(0 or False or 6) # 6
```
